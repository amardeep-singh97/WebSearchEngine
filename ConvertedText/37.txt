Template documentation[view] [edit] [history] [purge] This template is used on approximately 101,000 pages. To avoid major disruption and server load, any changes should be tested in the template's /sandbox or /testcases subpages, or in your own user subpage. The tested changes can be added to this page in a single edit. Consider discussing changes on the talk page before implementing them. This template should not be substituted. This template is used by one or more bots. If you intend to change this template in any significant way or nominate it for deletion, please inform the bot operators. Thank you. The relevant bots are: JJMC89 bot, QEDKbot and RussBot. Not to be confused with Template:Redirect category. For the Wikipedia policy on redirecting categories, see WP:CATRED.
