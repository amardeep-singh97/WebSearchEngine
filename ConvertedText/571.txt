The following categories are used to track and monitor this redirect: From outside category namespace: This is a redirect from a page outside the category namespace to a category page. This redirect page must be outside the category namespace. Redirects in category namespace are soft redirects, and rcats must never be used to tag soft redirects. Note: If this page is in category namespace, then it will populate Category:Pages with templates in the wrong namespace instead. From a shortcut: This is a redirect from a shortcut in any namespace to a page in any namespace except template namespace. Shortcuts are wikilinked on community pages, talk pages, and edit summaries, but not in mainspace articles. When appropriate, protection levels are automatically sensed, described and categorized.
